## 객체(Object)
- 여러 값을 하나의 묶음으로 관리하는 자료형
- key:value 형태의 프로퍼티(속성)로 구성
- 배열처럼 인덱스를 기준으로 하지 않고, 의미 있는 key를 기준으로 데이터를 관리

### 객체 생성
- 객체 생성 방식은 2가지(생성자, 리터럴)이며, 리터럴 방식을 많이 사용함

```js
let obj1 = new Object(); // 객체 생성자 방식
let obj2 = {}; // 객체 리터럴 방식
```

### 객체 프로퍼티(속성)
- key:value 구조로 작성
- 프로퍼티 개수 제한 없음
- key에 공백이 포함될 경우 문자열로 감싸야 함

```js
let person = {
    name: "장윤혁",
    age: 28,
    location: "분당",
    "like cat": true
};
```

### 프로퍼티 접근 방법
#### 점 표기법(dot notation)
- 가장 일반적이며 가독성이 좋음
- key가 유효한 식별자일 때만 사용 가능

```js
// 점 표기법
let name = person.name;
console.log(name); // 장윤혁
```

#### 괄호 표기법(bracket notation)
- key를 문자열로 지정
- 공백/특수문자가 포함된 key 접근 시 필수

```js
// 괄호 표기법
let age = person["age"];
console.log(age); // 28
```

#### 프로퍼티 추가
- 존재하지 않는 key에 값을 할당하면 자동 생성됨

```js
person.job = "fe developer";
person["favoriteFood"] = "삼겹살";
```

#### 프로퍼티 수정
- 기존 key에 새로운 값을 재할당 시 수정됨

```js
person.job  = "educator";
person["favoritefood"] = "회";

console.log(person.job); // educator
```

#### 프로퍼티 삭제
- delete 키워드 사용

```js
delete person.job;
delete person["favoriteFood"];

console.log(person.job); // undefined
console.log(person["favoriteFood"]); // undefined
```


#### 프로퍼티 존재 유무 확인
- "key" in 객체 형태로 확인
- 결과는 true/false 반환

```js
let result1 = "name" in person;
let result2 = "cat" in person;
console.log(result1); // true
console.log(result2); // false
```

## 상수 객체
- const로 선언한 객체는 재할당만 불가능함
- 객체 내부의 프로퍼티(추가·수정·삭제)는 가능

```js
const animal = {
    type: "고양이",
    name: "나비",
    color: "black"
};

animal.age = 2; // 추가
animal.name = "까망이"; // 수정
delete animal.color; // 삭제

// 단, animal 자체를 다른 객체로 재할당하는 것은 불가능
// animal = {} (에러)
```

## 메서드
- 값이 함수인 프로퍼티를 메서드라고 함

```js
const person2 = {
    name: "장윤혁",
    // 메서드 (함수 표현식)
    sayHi: function (){
        console.log("안녕하세요!")
    }
};
person2.sayHi(); // 안녕하세요!
```

## 화살표 함수
```js
const person3 = {
    name: "장윤혁",
    // 메서드 (화살표 함수)
    sayHi: ()=> {
        console.log("안녕하세요!")
    }
};
person3.sayHi(); // 안녕하세요! 
```

## 메서드 축약 표현
```js
const person4 = {
    name: "장윤혁",
    // 메서드 선언(축약 표현)
    sayHi() {
        console.log("안녕하세요!")
    }
};
person4.sayHi(); // 안녕하세요!
```
