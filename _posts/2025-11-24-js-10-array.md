## 배열(Array)
- 여러 개의 값을 순차적으로 담는 자료형
- 인덱스(index)를 이용해 요소에 접근
- 인덱스는 0부터 시작

### 배열 생성
- 배열 생성 방식은 2가지(생성자, 리터럴)이며, 리터럴 방식을 많이 사용함
- 숫자, 문자, 불리언, 함수 등 어떤 자료형도 담을 수 있음
- 요소 개수 제한 없음

```js
let arrA = new Array(); // 배열 생성자
let arrB = []; // 배열 리터럴

let arrC = [1, 2, 3, true, "hello", null, undefined, () => {}, {}, []];
```

### 배열 요소 접근
- 인덱스로 요소 접근 및 수정 가능

```js
let item1 = arrC[0];
let item2 = arrC[1];

arrC[2] = 4; // 요소 수정

console.log(item1, item2); // 1 2
console.log(arrC[2]); // 4
console.log(arrC.length); // 배열 길이 확인
```
