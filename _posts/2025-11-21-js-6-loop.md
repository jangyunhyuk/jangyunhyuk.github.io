## 반복문 (Loop)
- 반복 동작을 수행하기 위한 문법
- for, while 등이 대표적

```js
// 반복문 기본 구조
for (let idx = 0; idx < 3; idx++) { // for (초기식; 조건식; 증감식)
    console.log(idx) // 0
                     // 1
                     // 2
}
```
초기식
: - 반복문 안에서 사용할 변수를 처음 만들어서 초기화하는 부분
: - 이 안에서 선언한 변수는 반복문을 제어하는 역할을 하며, 보통 카운터 변수라고 부른다.

조건식
: - 반복문을 언제까지 반복할지 정해주는 식
: - 조건식의 결과가 참일 때만 반복이 진행

증감식
: - 반복이 한 번 끝날 때마다 실행되며, 카운터 변수의 값을 증가시키거나 변경해서 다음 반복으로 넘어가게 하는 부분

### break 문
- 중간에 강제 종료
```js
for (let idx = 1; idx < 10; idx++) {
    console.log(idx) // 1,2 출력되고 종료
    if (idx >= 2){
        break;
    }
}
```

### continue 문
- 반복문 안에서 특정 회차만 건너뛰고 다음 반복으로 넘어가고 싶을 때 사용

```js
for (let idx = 1; idx < 10; idx++) {
    if (idx % 2 === 0 ){
        // idx가 짝수면 아래 코드를 건너뛰고 다음 반복으로 넘어감
        continue;
    }
    console.log(idx) // 1,3,5 출력되고 종료
    if (idx >= 5){
        break;
    }
}
```
