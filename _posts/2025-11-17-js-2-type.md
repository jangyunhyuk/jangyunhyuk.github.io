## 자료형(Type)
자료형(Type) = 집합
: 동일한 속성이나 특징을 가진 원소들을 묶은 것<br>
(ex. 강아지와 고양이는 "동물" 이라는 집합으로 묶임)

### 원시타입
> 자료형에는 원시 타입과 객체 타입이라는 두 가지의 큰 부류로 나뉜다. (객체 타입은 추후에 포스팅 예정)

- Number, String, Boolean, Null, Undefined가 있음
- 기본형 타입이라고도 불림
- 프로그래밍에 있어 가장 기본적인 값들의 타입을 의미함

#### Number Type
- Number Type은 기본적인 사칙연산을 지원함

> 나머지 연산은 **모듈러 연산**이라고 함
{: .prompt-tip }

```javascript
let num1 = 27;
let num2 = 10;

console.log(num1 + num2); // 37
console.log(num1 - num2); // 17
console.log(num1 * num2); // 270
console.log(num1 / num2); // 2.7
console.log(num1 % num2); // 7
```

- 독특한 값들

```javascript
let inf = Infinity; // 양의 무한대
let mInf = -Infinity; // 음의 무한대

let nan = NaN; // not a number
// NaN은 수치연산을 실패했을 때의 결과 값
```

### String Type
- 문자열 값들은 덧셈 연산까지 지원함
- 템플릿 리터럴(Template Literal): 백틱(`)은 $와 {} 문법을 통해서 변수의 값을 동적으로 문자열에 포함시킬 수 있음

```javascript
let myName = "장윤혁";
let myLocation = "분당";
let introduce = myName + myLocation;
console.log(introduce); //장윤혁분당

// 템플릿 리터럴 문법
let introduceText = `${myName}은 ${myLocation}에 거주합니다.`
console.log(introduceText); // 장윤혁은 분당에 거주합니다.
```

### Boolean Type
- 참이거나 거짓만 저장하는 타입
- 상태를 표현하는 데 주로 사용
```javascript
let isSwitchOn = true;
let isEmpty = false;
```

### Null Type
- 변수에 어떠한 값도 담겨 있지 않음
```javascript
let empty = null;
console.log(empty); // null
```

### Undefined Type
- Undefined라는 하나의 값만 포함하는 특수한 타입
- 변수를 선언하고 그 변수에 어떠한 값도 집어넣지 않았을 때 자동으로 할당되는 값
```javascript
let none;
console.log(none); // undefined
```
